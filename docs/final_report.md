# kosLINK AI 최종 기획 및 개발 보고서 (Final Report)
> **자본시장법 규제 가이드라인을 완벽 준수하는 온톨로지 기반 소셜 인베스팅 플랫폼**

---

## 1. 프로젝트 요약 및 기획 배경 (Executive Summary)

### ① 기획 배경
- **반도체 공급망 정보의 폐쇄성**: 반도체 후공정 소켓 및 IP 설계 분야와 같이 전방 대기업(삼성전자, SK하이닉스)의 수혜를 입는 알짜 강소기업들의 실질 계약 관계는 공시 자료(DART) 깊숙이 숨어 있어 일반 개인 투자자가 직접 추적하기가 어렵습니다.
- **규제 위험성**: 개인이 포트폴리오를 짜서 유상으로 판매하거나, AI가 유저의 자산 상태를 기반으로 실시간 맞춤형 ETF 종목을 자의적으로 가중 추천하는 행위는 모두 **자본시장법 제6조(미등록/미신고 투자자문업)**에 저촉되는 불법 행위입니다.

### ② 플랫폼 솔루션
- `kosLINK AI`는 신뢰할 수 있는 금감원 공시 정보를 가공한 **3-Layer 밸류체인 온톨로지**를 시각화하여 정보 비대칭성을 해소합니다.
- 자본시장법 준수를 위해 **'무연산 결정론적 룰 엔진'**과 **'대가성 없는 소셜 피드 공유(투자 SNS) 모델'**을 결합하여 금융 규제 리스크를 기술적으로 원천 제거하고, 합법적인 B2B/B2C 비즈니스 모델과 크리에이터 기여도 보상 펀드를 수립했습니다.

## 2. 서비스 핵심 아키텍처 및 화면 구성

본 플랫폼은 PC 브라우저 와이드 뷰 환경에 최적화된 3단계 독립 탭 구조로 설계되었습니다.

### ① [TAB 1] AI 챗봇 빌더 (Graph-RAG & Rule-based Slot)
- **자연어 팩트 질의**: 유저가 던진 밸류체인 질문에 대해 Graph-RAG 엔진이 팩트 데이터만을 추출하여 답변합니다.
- **결정론적 비중 계산**: 복잡한 투자 연산 없이, 온톨로지 상의 종목 체급 속성에 따라 고정된 비율(중형주 40%/35%, 소형주 25%)로 자산 배분 비중을 계산하여 우측 접이식 패널에 실시간 렌더링합니다.

### ② [TAB 2] 3-Layer 밸류체인 온톨로지 (Interactive SVG)
- **3-Layer 매핑**: `[테마 분야: 온디바이스 AI]` ➔ `[밸류체인 역할: 칩 제조, 검사부품, IP설계]` ➔ `[상장 종목]`의 3단계 흐름을 SVG 실시간 연동선으로 표현합니다.
- **종목 간 SUPPLY_TO(공급망) 시각화**: 리노공업, ISC, 오픈엣지가 최종 전방 제조사인 **삼성전자**로 수렴하는 계약 노선을 **오렌지색 점선**으로 연결하고, 팩트 캡슐 마우스 호버 시 Glowing 강조 애니메이션을 제공합니다.
- **심사위원 명분 패널**: 할루시네이션(환각)을 완벽 차단하는 원리와 금융 연산 오류를 우회한 룰 엔진 논리를 기술적으로 설득합니다.

### ③ [TAB 3] 소셜 인베스팅 커뮤니티 피드 (Social Sharing Feed)
- **집단지성 피드**: 유저들이 MCP 프로토콜로 공유한 무료 커뮤니티 지수들이 추천수(Like)와 함께 나열됩니다.
- **기간별 수익률 필터**: 1주, 1개월, 6개월, 1년 기간 필터를 스위칭하여 국내 거래소 컬러 규격(상승: 빨간색, 하락: 파란색)으로 수익률Badge를 실시간 전환 갱신합니다.
- **상세 분석 모달**: 개별 ETF 클릭 시 세부 종목 비중 수평 바 차트와 함께 **챗봇 분석을 RAG 요약한 해당 ETF의 핵심 장점**을 팝업으로 조회합니다.

## 3. 자본시장법 준수를 위한 규제 극복 전략 (Compliance)

### ① 미등록 투자자문업 원천 차단
- 유저의 개인 프로파일링 정보(자산, 나이, 위험 성향)를 일절 수집하지 않음으로써 '개별 맞춤 상담' 성격을 법적으로 배제합니다.
- AI가 매수/매도 시점이나 가치 판단(전망)을 언급하지 못하도록 시스템 프롬프트(가드레일)를 치고, 오직 객관적 팩트 분석 및 쏠림 리스크(검사부품에 75% 편중 등)만 알려주는 **'정량적 포트폴리오 건강 검진(Diagnostics)'** 포맷만 제공합니다.

### ② 유사투자자문업 가이드라인 준수
- 유저 간 유료(실제 현금) 지수 판매 모델은 불법이므로, 플랫폼 내의 모든 거래 및 공유는 **'무상 공유(SNS)'** 또는 **'가상 포인트 시스템'**으로 제한합니다. 대가성과 영업성이 없으므로 표현의 자유 및 정보 공유로 인정받아 안전합니다.

## 4. 실시간 온톨로지 동기화 및 자동화 파이프라인 (Data Pipeline)

정적 온톨로지의 한계를 해소하기 위해 제안된 공공 API 기반의 실시간 정보 적출 파이프라인 아키텍처입니다.

```
[ Open DART API / RSS ] ➔ [ 텍스트 추출 및 전처리 ] ➔ [ LLM JSON 파싱 (Gemini) ]
                                                              │
[ Neo4j Graph DB ] 💻 <─── [ 리서처 최종 승인 ] ─── [ HITL Curation 대시보드 ]
```

1. **상장 종목 시총 동기화**: 매일 장 마감 후 KRX 상장종목 API를 스케줄러로 기동하여 시가총액 변동에 따른 종목 노드의 **{체급} 속성**을 자동 갱신합니다.
2. **공시 계약 트래킹**: 1시간 주기로 금감원 Open DART API를 폴링해 반도체 공급 계약 공시를 트래킹합니다.
3. **LLM Structured Extraction**: 감지된 공시 문서에서 LLM(Gemini)이 JSON 스키마 규격에 맞게 공급사, 수요처, 제품명, 계약금액을 개체(Entity)로 자동 추출합니다.
4. **HITL (Human-in-the-loop) Curation**: LLM의 추출 에러나 환각을 막기 위해 리서처가 승인/반려할 수 있는 어드민 검증 단계를 거쳐 최종 Neo4j Graph DB에 반영합니다.

## 5. 지속 가능한 비즈니스 모델 (BM) 및 유저 인센티브

### ① 합법적인 3대 비즈니스 모델 (BM)
- **B2C 프리미엄 구독 멤버십**: 실시간 DART 공시 감지 문자 알림, 고급 밸류체인 PDF 요약본 제공, 챗봇 무제한 질문 이용권(데이터 단말기 이용료 징수 모델).
- **B2B 인덱스 공급 라이선싱 (OEM)**: 소셜 피드에서 집단지성으로 검증된 우수 인덱스 IP를 선별하여 제도권 자산운용사에 ETF 상품 지수로 공급하고 사용료(Royalty Fee)를 수취.
- **증권사 연동 아웃링크 수수료**: 유저들이 마음에 드는 ETF를 제휴 증권사 계좌로 연동해 즉시 원클릭 바스켓 주문하도록 중개하고 아웃링크 Commission 획득.

### ② 유저들의 무료 공유 유인을 위한 인센티브
- **크리에이터 기여도 펀드 (Creator Fund)**: 플랫폼이 구독료와 B2B 라이선스 수수료로 얻은 매출 일부를 활용해, 공유 피드에서 성과 지표(조회수, 추천수, 백테스트 수익률)가 우수한 ETF 제작 유저에게 크리에이터 활동 보상금을 정산 지급(AdSense 모델).
- **소셜 트랙레코드 명성 및 리더보드**: 매일 모의 수익률을 랭킹표에 게시해 퀀트 연구원으로서의 트랙레코드 명성을 공증받게 함으로써 개인 브랜딩 수단으로 활용 유도.
- **AI 고급 진단 크레딧 보상**: 본인의 인덱스를 피드에 전체 공개로 공유할 때마다 챗봇 빌더의 API 이용권 크레딧을 무료로 보상.

## 6. MCP (Model Context Protocol) 연동 모델

챗봇과 소셜 피드를 유기적으로 연결하기 위해 MCP 클라이언트-서버 프로토콜 스펙을 연동합니다.

- **Search Loop**: 사용자의 자연어 입력("HBM3E 지수 찾아줘")을 가로채 AI가 **`search_community_indices()`** 도구를 호출해 관련 피드를 렌더링(검색 대행).
- **Publish Action Loop**: 챗봇 대화 중 포트폴리오를 드래프트 저장 및 공유 지시 시 AI가 **`publish_social_index()`** 도구를 호출해 커뮤니티 피드에 포스팅하고, 프론트엔드가 이를 받아 오버레이 연출 후 피드 탭으로 화면 자동 라우팅 처리.
- **Diagnosis Resource Loop**: 소셜 피드에서 개별 카드의 분석 의뢰 시 `index://<id>` **Resource URI**를 챗봇 프롬프트에 주입하여 즉각적인 공시 팩트 기반 진단 보고서를 출력.

- **데브옵스 (1인)**: NCP 금융망 인프라 구축, Terraform 관리, ArgoCD GitOps 자동 배포.
- **프론트엔드 (1인)**: UI/UX, SVG 3-Layer 온톨로지 드로잉, 탭 라우팅, 피드 모달 구현.
- **AI 백엔드 (2인)**: Open DART 데이터 파이프라인 개발, LLM JSON 개체명 적출, Neo4j Graph-RAG 구성.
- **소셜 백엔드 (2인)**: 소셜 피드 DB 모델링, 모의 수익률 연산 엔진, MCP 호스트 서버 구축, 크리에이터 기여도 보상 스케줄러 개발.
