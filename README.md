# kosLINK AI 🚀
> **3-Layer 밸류체인 온톨로지 기반의 AI 인덱스 빌더**

`kosLINK AI`는 국내 반도체 공급망 데이터를 활용하여 자산운용 및 리서치 검증 프로세스를 혁신하는 프리미엄 핀테크 SaaS 서비스입니다. 금감원 가이드라인 및 자본시장법 규제를 준수하기 위해 구조화된 **3-Layer 밸류체인 온톨로지**와 **Graph-RAG 챗봇**을 연동하여, 환각 없는 안전한 투자 포트폴리오(ETF) 구성을 지원하고 이를 소셜 커뮤니티(SNS)에 공유 및 소통할 수 있게 합니다.

---

## 📌 핵심 기능 (Key Features)

### 1. 🔮 AI 챗봇 빌더 (RAG & Rule-based Weighting)
- **규제 최적화 필터링**: 개인 인적 사항의 자동 수집 제외 및 찌라시 테마주 추천 방어를 위해 DART 공시 기반으로 입증된 종목만 추천 풀에 편입합니다.
- **무연산 비중(%) 산출**: 복잡한 연산 공식 없이, 온톨로지 상의 종목 체급 속성(대형주 소거 ➔ 중형주 40%/35%, 소형주 25%)에 입각해 ETF 비중 가이드를 실시간 도출합니다.
- **다변화된 시연 가이드**: 
  - *시연 1 (포트폴리오 빌드)*: ETF 생성 질문 입력 및 우측 실시간 미러 패널 구성 연동.
  - *시연 2 (밸류체인 질의)*: 리노공업 등 특정 기업의 밸류체인상 역할 및 전방 산업 관계 정보 텍스트 응답 제공.

### 2. 🗺️ 3-Layer 밸류체인 온톨로지 (Graph DB UI)
- **3-Layer 구조화**: `[사업 분야 (테마)]` ➔ `[밸류체인 역할 (기술)]` ➔ `[상장 종목]`으로 분류되는 직관적 흐름을 제공합니다.
- **SUPPLY_TO(공급 관계) 매핑**: 리노공업, ISC, 오픈엣지가 최종 전방 제조사인 **삼성전자**로 수렴하는 계약/공급망 라인을 점선(Dashed Line)으로 시각화했습니다.
- **DART 팩트 가드레일**: 관계선 위에 금감원 DART 및 사업보고서 공시 팩트 캡슐을 상시 배치하여 신뢰성을 확보하고, 호버 시 해당 라인을 강조 표시합니다.
- **심사위원 설득 명분 패널**: 할루시네이션 방어 원리와 금융 연산 최소화 비중 도출 메커니즘을 상세히 설명하는 카드 패널을 우측에 상시 기재했습니다.

### 3. 👥 소셜 인베스팅 커뮤니티 피드 (Social Sharing)
- **대가성 없는 무료 공유**: 개인 자문 위반 법적 리스크 차단을 위해 원화 거래를 배제하고, 추천수(Like)와 가상 크레딧 및 백테스팅 기반의 투자 커뮤니티로 작동합니다.
- **기간별 수익률 필터**: `1주`, `1개월`, `6개월`, `1년` 필터를 장착하여 국내 거래소 규격(상승: 빨간색, 하락: 파란색)으로 수익률 추이를 실시간 전환 추적합니다.
- **ETF 상세 정보 조회 모달**: 개별 ETF 카드를 선택하면 세부 자산 구성 비중 그래프와 함께 **챗봇 대화 기반 특징/장점 요약(AI 팩트체크 리포트)**이 팝업으로 노출됩니다.
- **MCP 원격 소셜 피드 공유**: 챗봇 빌더에서 생성 완료한 인덱스를 커뮤니티 피드 데이터베이스에 원격 발행하고 강제 탭 전환 연동을 구현했습니다.

## 🐳 실행 방법 (How to Run)

### 1. 로컬 개발 환경 실행
Nginx 서버나 VS Code Live Server 등을 이용해 `index.html` 파일을 즉시 브라우저로 오픈하여 구동할 수 있습니다.

### 2. Docker를 활용한 구동
1. **도커 이미지 빌드**:
   ```bash
   docker build -t test-web:latest ./test-web-app
   ```
2. **컨테이너 실행**:
   ```bash
   docker run -d -p 8080:80 --name test-web-container test-web:latest
   ```
3. **웹 접속**:
   브라우저에서 `http://localhost:8080` 경로로 접속하여 시연을 시작합니다.
